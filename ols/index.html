What is OLS?

Ordinary Least Squares (OLS) is a method for estimating the unknown parameters in a linear regression model.

It minimizes the Sum of Squared Residuals (SSR). A residual \(e_i\) is the vertical distance between the actual data point and the regression line.

\[ e_i = y_i - \hat{y}_i \] \[ \text{SSR} = \sum_{i=1}^n e_i^2 = \sum_{i=1}^n (y_i - (m x_i + c))^2 \]

Squaring the residuals ensures positive and negative errors don't cancel out, and penalizes larger errors more heavily.

Scaling to N Dimensions

By writing each linear equation \(y_i = \beta_0 + \beta_1 x_i + \epsilon_i\), we can stack all observations into a single elegant system:

\[ \mathbf{y} = \mathbf{X}\boldsymbol{\beta} + \boldsymbol{\epsilon} \]

Notice that the first column of the Design Matrix \(\mathbf{X}\) consists entirely of 1s. This is the bias / intercept column, which gets multiplied by \(\beta_0\).

The second column contains your raw \(x_i\) features, which gets multiplied by \(\beta_1\).

Step 5 — Solving the Normal Equation

In matrix form, the OLS solution is computed in a single step using the Normal Equation. Watch every intermediate matrix operation calculated live.

1. Transpose design matrix product: \(\mathbf{X}^T\mathbf{X}\)
\(2 \times 2\)
\[ \mathbf{X}^T\mathbf{X} = \begin{pmatrix} n & \sum x_i \\ \sum x_i & \sum x_i^2 \end{pmatrix} \]

Holds sample size, sum, and sum of squares of \(x\).

2. Invert the product: \((\mathbf{X}^T\mathbf{X})^{-1}\)
\(2 \times 2\)
\[ \mathbf{A}^{-1} = \frac{1}{ad - bc}\begin{pmatrix} d & -b \\ -c & a \end{pmatrix} \]

Computed using the analytical 2D matrix inverse. Determinant = 0.00.

3. Transpose design matrix and output: \(\mathbf{X}^T\mathbf{y}\)
\(2 \times 1\)
\[ \mathbf{X}^T\mathbf{y} = \begin{pmatrix} \sum y_i \\ \sum x_i y_i \end{pmatrix} \]

Projecting the target vector \(\mathbf{y}\) onto the column space of \(\mathbf{X}\).

4. Multiply to solve: \(\boldsymbol{\beta} = (\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T\mathbf{y}\)
\(2 \times 1\)

This gives the exact same OLS coefficients as the simple analytical formulas in Step 2!

\(\beta_0 = \) 0.00
\(\beta_1 = \) 0.00

The Derivation

To find the \(\boldsymbol{\beta}\) that minimizes the squared residuals, we write the objective function in matrix form:

\[ S(\boldsymbol{\beta}) = (\mathbf{y} - \mathbf{X}\boldsymbol{\beta})^T(\mathbf{y} - \mathbf{X}\boldsymbol{\beta}) \]

Taking the gradient with respect to \(\boldsymbol{\beta}\):

\[ \nabla_{\boldsymbol{\beta}} S = -2\mathbf{X}^T\mathbf{y} + 2\mathbf{X}^T\mathbf{X}\boldsymbol{\beta} \]

Setting the gradient to \(\mathbf{0}\) yields the Normal Equation:

\[ \mathbf{X}^T\mathbf{X}\boldsymbol{\beta} = \mathbf{X}^T\mathbf{y} \] \[ \boldsymbol{\beta} = (\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T\mathbf{y} \]

Step 6 — OLS Assumptions & Diagnostic Plots

Linear regression relies on key assumptions (LINE) to make valid statistical inferences. Examine these diagnostic charts calculated live from your data.

Residuals vs. Fitted Values

Tests Homoscedasticity (equal variance) and Linearity. Points should be scattered randomly around the 0 line with no distinct pattern.

Residual Distribution Histogram

Tests Normality of Errors. Residuals should ideally follow a bell-shaped normal curve centered at 0.

Evaluating Model Fit

Coefficient of Determination (\(R^2\))

Measures the proportion of variance in the dependent variable \(y\) that is predictable from the independent variable \(x\).

\[ R^2 = 1 - \frac{SS_{\text{res}}}{SS_{\text{tot}}} = \mathbf{\text{loading...}} \]

Adjusted \(R^2\)

Adjusts \(R^2\) based on the number of predictors. Prevents overfitting inflation.

\[ R^2_{\text{adj}} = 1 - (1 - R^2)\frac{n-1}{n-p-1} \] \[ R^2_{\text{adj}} = \mathbf{\text{loading...}} \]

The LINE Assumptions Checklist

  • L · Linearity: Is the relationship linear? Check the Residuals vs Fitted plot; a curved trend implies non-linearity.
  • I · Independence: Are errors independent? Ensure observations don't have auto-correlation.
  • N · Normality: Are errors normally distributed? Check the residual histogram distribution.
  • E · Equal Variance (Homoscedasticity): Are residuals spread equally? A funnel shape in Residuals vs Fitted implies heteroscedasticity.
